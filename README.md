🕹️ How to Use

Select Algorithm — choose A* or Greedy BFS from the dropdown
Select Heuristic — choose Manhattan or Euclidean
Set up the grid — either draw walls manually by clicking/dragging, or click ⚡ Generate Maze for a random map
Click ▶ Run Agent — watch the agent find its path in real time
Click ↺ Reset — clear the board and start over
Optionally change grid size using the Rows / Cols inputs and click Resize Grid
